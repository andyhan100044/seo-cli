## 🎯 常用操作示例

### 示例1：发现今天热门的关键词

```bash
python seo.py discover --limit 20
```

**作用：** 自动找出当前最热门的20个关键词

**结果：** 会生成一个CSV文件，包含关键词、搜索量、热度分数

---

### 示例2：深度分析一个关键词

```bash
python seo.py intent --word "简历模板" --longtail 30
```

**作用：** 分析"简历模板"这个关键词
- 自动判断搜索意图
- 生成30个相关长尾词
- 给出网站建设建议

**结果：** 生成2个文件
- JSON文件：详细数据
- Markdown文件：网站建设规划

---

### 示例3：批量分析多个关键词

**步骤1：** 创建一个文本文件 `keywords.txt`，内容如下：
```
AI工具
在线翻译
图片压缩
视频剪辑
```

**步骤2：** 运行命令
```bash
python seo.py batch --file keywords.txt --output results.csv
```

**作用：** 同时分析文件中的所有关键词

**结果：** 生成一个CSV文件，包含所有关键词的分析结果

## 💡 使用技巧

### 1. 关键词选择技巧
**好关键词的特征：**
- 明确的需求（如"简历模板下载"）
- 有商业价值（如"AI工具"）
- 竞争不过于激烈

**避免的关键词：**
- 过于宽泛（如"电脑"）
- 没有明确意图（如"怎么样"）

---

### 2. 结果解读技巧

**意图类型说明：**
- **transactional（交易意图）** → 用户想购买或使用，适合做工具站
- **informational（信息意图）** → 用户想了解，适合做博客
- **navigational（导航意图）** → 用户想找网站，适合做导航站

**热度分数：**
- 70+：非常热门
- 40-69：较热门
- 20-39：一般热门
- <20：冷门

---

### 3. 效率提升技巧

1. **批量处理**：用 `batch` 命令一次分析多个关键词
2. **保存结果**：定期备份结果文件
3. **对比分析**：对比不同时间的结果，发现趋势
4. **组合使用**：先用 `discover` 发现热词，再用 `intent` 深度分析
